This page holds nothing on its own. Your plan and tick-offs live in a private GitHub repo that only your token can read, so the same progress follows you across devices and nobody else can open it.

A fine-grained token limited to your one data repo, with Contents: read and write. Create one → Set an expiry; revoking it stops access on every device at once.
The private repo that holds your plan, as owner/name. Keep this separate from the public repo serving this page.
Pick your plan-data.json. It is written into the private data repo above, never into your public Pages repo.
  1. Create an empty private repo on GitHub for your data (e.g. training-log-data).
  2. Make a fine-grained token limited to that repo, with Contents: read and write.
  3. Enter the token and owner/name above, choose your plan-data.json, and connect.
  4. On any other device, switch to “I already set this up” and enter the same token and repo.
